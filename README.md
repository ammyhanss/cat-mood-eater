# 🐈 Cat Mood Eater

一個專為打工人與高壓族群設計的 macOS 桌面情緒垃圾桶。
將煩惱寫下並交給虛擬橘貓吃掉，透過具象化的互動設計完成情緒轉換。

## ✨ 特色亮點

* 🐈 **專屬情緒宣洩口** - 常駐桌面的輸入介面，隨時記錄日常遇到的 Bug 或負面情緒。
* 🎬 **療癒互動動畫** - 點擊「要出氣！」按鈕，觸發專屬的碎紙與橘貓吞食視覺效果。
* 📊 **情緒計數統計** - 自動記錄「今天吃掉了幾個負面情緒」，讓壓力釋放過程具象化。
* 💻 **輕量級桌面組件** - 原生風格的小組件設計，不干擾日常工作流程。

## 🚀 快速開始

### 前置要求

在本地運行此專案前，請確保你的 macOS 已安裝以下環境：
* [Node.js](https://nodejs.org/) (建議版本 16.0 或以上)
* npm (通常隨 Node.js 一併安裝)

### 安裝與運行

打開 macOS 的「終端機 (Terminal)」，依序執行以下指令：

```bash
# 1. 克隆專案到本地
git clone [https://github.com/ammyhanss/cat-mood-eater.git](https://github.com/ammyhanss/cat-mood-eater.git)

# 2. 進入專案目錄
cd cat-mood-eater

# 3. 安裝所需依賴套件
npm install

# 4. 啟動桌面小組件
npm start
